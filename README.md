# CareerConnect – Student Job Portal Dashboard

## Overview

CareerConnect is a responsive web application designed to help students explore and apply for job opportunities. The platform allows students to create a profile, search and filter job listings, apply for jobs, and track their applications.

## Features

* Student login and profile management
* Job search functionality
* Job category filtering
* Apply for jobs
* Applied jobs tracking
* Dark mode support
* Responsive design for mobile and desktop devices
* Data persistence using browser Local Storage

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Browser Local Storage

## How to Run the Project

1. Download or clone the repository.
2. Open the project folder in Visual Studio Code.
3. Run the application using the Live Server extension or open `index.html` in a web browser.
4. Enter your details and log in to access the dashboard.

## Application Workflow

1. Students enter their name, branch, and skills on the login page.
2. Profile information is stored in Local Storage.
3. Students can browse available jobs.
4. Jobs can be searched and filtered by category.
5. Students can apply for jobs and track their applications.
6. Applied jobs and theme preferences are stored locally.
